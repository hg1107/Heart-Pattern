# Heart Shape Animation (Python Turtle)

## Project Description

This project creates a heart-shaped graphic using Python.
The notebook demonstrates graphical drawing techniques, coordinate movement, and curve formation using Python’s graphics capabilities (commonly Turtle or similar drawing libraries).

It is designed as a simple visual animation project to demonstrate shape construction and basic graphical logic.

## How to Run

1. Install Python 3.
2. Install Jupyter Notebook (if not installed):

   ```
   pip install notebook
   ```
3. Run:

   ```
   jupyter notebook
   ```
4. Open heart.ipynb.
5. Run all cells to display the heart graphic.

---

## How It Works

The program:

1. Initializes the drawing window.
2. Positions the cursor.
3. Uses movement and rotation commands to create curved arcs.
4. Combines curves and lines to form a heart shape.
5. Fills the shape with color (if implemented).
6. Keeps the window open for display.

The heart is formed mathematically using rotations and arc-like curves.
